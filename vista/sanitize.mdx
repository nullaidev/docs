Sanitize HTML from using `\Nullai\Vista\SanitizeHtml::escHtml()` and `\Nullai\Vista\SanitizeHtml::escAttr()`:

```php
<?php
/**
 * Sidebar view file
 * @var array $menu The array of menu items passed to this view
 * @var $this \Nullai\Vista\Engines\ViewRenderEngine
 */
?>
<aside class="sidebar">
    <nav class="menu">
        <ul>
            <?php foreach ($menu ?? [] as $item): ?>
                <li>
                    <a href="#" title="<?= \Nullai\Vista\SanitizeHtml::escAttr($item) ?>">
                        <?= \Nullai\Vista\SanitizeHtml::escHtml($item) ?>
                    </a>
                </li>
            <?php endforeach; ?>
        </ul>
    </nav>
</aside>
```

## Sanitize JSON

Sanitize data and encode it as JSON using `\Nullai\Vista\SanitizeHtml::escJson()`:

```php
<?php
/**
 * Sidebar view file
 * @var array $menu The array of menu items passed to this view
 * @var $this \Nullai\Vista\Engines\ViewRenderEngine
 */
?>

<?php $this->section('scripts'); ?>
<script>
    // Safely embed JSON in a JavaScript variable
    console.log(<?= \Nullai\Vista\SanitizeHtml::escJson(['site' => '<My Site>']) ?>);
</script>
<?php $this->end(); ?>
```

## Whitelist & Blacklist HTML Tags

Sanitize and allow specific HTML tags and attributes with `\Nullai\Vista\SanitizeHtml::allowTags()`:

```php
<?php
echo \Nullai\Vista\SanitizeHtml::allowTags(
    html: '<p><a href="#" class="underline">Link</a><p>',
    tags: 'a:href,class,p',
    allow: true
);
```

Blacklist tags:

```php
<?php
echo \Nullai\Vista\SanitizeHtml::allowTags(
    html: '<iframe><iframe><script><script>',
    tags: 'iframe,script',
    allow: false
);
```
