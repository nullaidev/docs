---
title: 'Filter HTML'
description: 'Whitelist and blacklist HTML tags.'
icon: 'filter'
---

Filter specific HTML tags and attributes with `\Nullai\Hygiene\SanitizeHtml::allowTags()`:

```php
<?php
echo \Nullai\Hygiene\SanitizeHtml::allowTags(
    html: '<p><a href="#" class="underline">Link</a><p>',
    tags: 'a:href,class,p',
    allow: true
);
```

Blacklist tags:

```php
<?php
echo \Nullai\Hygiene\SanitizeHtml::allowTags(
    html: '<iframe><iframe><script><script>',
    tags: 'iframe,script',
    allow: false
);
```
